# aditya5338.github.io
My personal portfolio — built with vanilla HTML, CSS &amp; JS. Features dark/light mode, scroll animations, and a page loader.

Aditya Kumar — Portfolio

Hey! This is my personal portfolio website. I built it from scratch using just HTML, CSS and JavaScript — no frameworks, no templates.

🌐 **Live at:** [aditya5338.github.io](https://aditya5338.github.io)

What's inside

- Dark & light mode (remembers your preference)
- Page loader + scroll animations
- Sections for my projects, experience, skills and contact
- Fully responsive — works on mobile too
- Single HTML file — no build tools, no npm, nothing
